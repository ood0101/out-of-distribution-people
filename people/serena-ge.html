Research & Key Work

Problems & Research Areas

  • High-quality code training data for LLMs
  • Supervised fine-tuning data for coding workloads
  • Reinforcement learning environments for code
  • LLM reasoning capabilities
  • Synthetic data generation

Key Work

Co-founded Datacurve, which provides frontier coding data for training and evaluating LLMs using a 'bounty hunter' system for skilled engineers. Previously improved LLM reasoning capabilities at Cohere.

Companies Founded

  • Datacurve

Current Work

CEO of Datacurve, building the data infrastructure layer for frontier AI model training, starting with code.

◆ Alpha Insight

Identified that the biggest bottleneck for vertical LLM capability is curated high-quality training data, not model architecture. The bounty-hunter model for sourcing expert data could expand far beyond code into finance, medicine, etc.
