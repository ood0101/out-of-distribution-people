◆ Investment Thesis

Sijun Tan is building the de facto open-source infrastructure layer for RL post-training of LLM agents — the picks-and-shovels play for every company that wants to make AI agents better through reinforcement learning. rLLM has 5,300+ GitHub stars, 60 contributors, 2M+ HuggingFace model downloads, and is already being used by Alibaba (Tongyi DeepResearch), Snorkel AI, and Together AI. His co-lead Michael Luo graduated in Dec 2025, went to DeepMind, and is now building a stealth startup — a sign the Agentica team is transitioning from lab project to company. Sijun is still in his PhD (expected 2027), but the project’s momentum — shipping SOTA models every 2 months, a new Hive platform (Mar 2026), and SDK integrations with every major agent framework — suggests the commercialization window is open. The angle: build the relationship now through the rLLM open-source community before the inevitable spinout.

The Spike — With Evidence

Sijun Tan is the #1 contributor to rLLM (by commit count on GitHub) and first author on the rLLM framework paper. The framework has produced three SOTA-class models in rapid succession:

The key innovations are technical, not just scale: GRPO+ (an improved GRPO algorithm with offline difficulty filtering, overlong filtering from DAPO, no KL/entropy loss), iterative context lengthening (8K→16K→24K, training models to “think shorter, then longer”), and one-off pipelining (2x speedup over baseline RL training via verl-pipeline). DeepScaleR required only 3,800 GPU hours — an 18x cost reduction over prior approaches. DeepCoder was trained in 2.5 weeks on 32 H100s.

The rLLM SDK (Dec 2025) enables any agent framework (LangGraph, SmolAgent, Strands, OpenAI Agents SDK, Google ADK) to be trained with RL via a single decorator — zero code changes to production agent code. This is the abstraction that makes rLLM a platform, not just a research project.

Earlier work shows systems depth: CryptGPU (IEEE S&P 2021, 278 citations) achieved 6–36x speedup for privacy-preserving ML on GPUs. MPCAuth (IEEE S&P 2023) and Flock (OSDI 2024) demonstrate a rare breadth spanning security, distributed systems, and ML.

What He’d Build

The RL Post-Training Infrastructure Thesis

rLLM is already a platform: it supports any agent framework (LangGraph, SmolAgent, OpenAI Agents SDK, Google ADK), multiple RL algorithms (GRPO, REINFORCE, RLOO, rejection sampling), two training backends (verl for multi-GPU, tinker for single-machine), 50+ built-in benchmarks, and a CLI-first workflow. The SDK requires near-zero code changes to make production agents trainable. This is the “Weights & Biases for agent RL” or “Hugging Face TRL on steroids” play.

The newest product, Hive (Mar 2026), is a collaborative platform where AI agents evolve shared artifacts — agents register, clone tasks, submit improvements to leaderboards, and share skills. This adds a marketplace/platform dynamic on top of the training infrastructure.

Market

  • AI agent infrastructure: estimated $15B+ by 2028
  • RL/RLHF post-training services: every frontier lab and AI company needs this capability
  • The “training tax” on agent builders is enormous — rLLM showed a 4B model can outperform 235B with the right RL recipe, representing 50x+ compute savings
