What Modal Does

Modal is a serverless cloud platform for AI/ML workloads. Developers deploy Python functions that auto-scale from zero to thousands of GPUs with a single decorator. Custom-built infrastructure stack: own file system, container runtime (sub-second cold starts), scheduler, image builder. Supports NVIDIA H100, A100, L40S, A10, L4, T4. Usage-based pricing (pay only for execution time). Use cases: AI inference, LLM fine-tuning, batch processing, code sandboxes, computational biotech.

Key technical differentiator: Modal built every layer of the AI infra stack from scratch. Not a wrapper on AWS/GCP. Custom container runtime enables sub-second startup. This is infrastructure-level engineering, not application-layer convenience — the kind of thing only an IOI gold medalist + quant + Scale AI veteran would attempt.

Risks

  1. GPU commoditization & pricing pressure. RunPod offers H100s at $2.39/hr vs Modal ~$3.95/hr. As GPU supply increases (NVIDIA B200 ramp, AMD MI300X), the hardware margin compresses. Modal’s moat must be developer experience, not compute pricing
  2. Hyperscaler competition. AWS, GCP, and Azure are all building serverless GPU offerings. Google Cloud Run added GPU support in 2025. If a hyperscaler replicates Modal’s DX at scale, the standalone infra play gets harder
  3. Revenue concentration risk. If a handful of large AI companies (Cognition, Cursor, Suno) drive a disproportionate share of ~$50M ARR, customer churn or in-housing could impact growth
  4. $2.5B valuation at ~$50M ARR = 50x revenue multiple. Aggressive even for AI infra. Requires continued hypergrowth to justify. Any slowdown in AI inference demand growth could compress the multiple
  5. Two-person founding team balance. Erik (CEO) and Akshat (CTO) is a strong pairing, but Modal’s growth to 118 people means organizational complexity. CTO-to-VP-Eng transition is a common scaling challenge
