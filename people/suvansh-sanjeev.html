Technical Deep Dive

MS Thesis: Quadrotor Dynamics (CMU, Dec 2022)

Learning Parameter-Efficient Markovian Quadrotor Dynamics Models (CMU-RI-TR-22-68). The problem: standard physical models work for hovering quadrotors but fail during aggressive maneuvers where aerodynamic effects (lift, drag, propeller interactions, downwash) dominate. Suvansh’s solution: a two-stage approach — (1) fit an 18-parameter symmetric quadratic function for aerodynamic forces, then (2) augment with a learned d-dimensional latent state. Result: a Markovian state 16x smaller than sliding-window approaches with strong trajectory fitting. This is gray-box ML — physics-informed learning, not pure neural network curve-fitting.

Safety Guarantees (ICRA 2021, with Tomlin)

Hamilton-Jacobi reachability analysis for computing safety envelopes for autonomous systems. The “scalable” contribution is making HJ methods work on higher-dimensional systems where traditional grid-based approaches are computationally intractable. This is foundational safety work for any autonomous system (drones, self-driving, surgical robots).

Language-Guided RL (ICLR 2019)

Using natural language instructions to guide RL policy learning via meta-learning. The agent learns to follow language commands without being pre-programmed for specific instructions. This was prescient: the idea that language models can serve as the interface between humans and autonomous agents is now the dominant paradigm in robotics (see: SayCan, RT-2, etc.).

Ecological RL (NeurIPS Workshop 2019)

RL in non-episodic settings: the agent must learn through continuous interaction without resets, with only delayed/sparse rewards. This is closer to how real robots operate in the real world. The paper argues the episodic RL paradigm (reset, try again) is unrealistic for deployment.
